什麼是變數

可以將變數想像成「一個能夠裝東西的箱子」，我們可以幫這個箱子命名（例如：box），箱內能儲存一些東西，這樣就可以用我們命名的 box 來代指箱子裡所儲存的東西。

undefined = 有宣告，但沒有賦值
not defined = 尚未宣告
