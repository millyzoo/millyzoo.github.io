JavaScript - this

2月 28 2021 Front-End

在不是物件導向的環境下，this 的預設值會是 global，依據不同環境呼叫 this，所得到的值也會不同。

例如：在 node.js 裡面呼叫預設值會是 global 的東西，在瀏覽器上則會得到 windows 的東西。

是不是聽起來覺得很奇怪？ this 的值為什麼不是我們想要的值？
